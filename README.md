
# create-404kit

一个快速生成项目搭建环境的脚手架

## 使用

```bash
pnpx create-404kit

# 或者

npx create-404kit

# 或者

bun create 404kit

# 或者

pnpm create 404kit
```

## 开发

```bash
bun run build

# publish
bunx npm login

bun publish

# new version
bunx npm version patch

# 发布
bun publish
```

- [Taro 4 基础脚手架](https://github.com/initH271/taro-react-template#main)
